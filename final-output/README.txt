This is the archive that contains the final project for group 13.

BAIS6040-project-group13 archive contains the following:
BAIS6040-group13-final-dataprep.ipynb - data gathering and cleaning notebook
BAIS6040-group13-final-modeling.ipynb - model building and model export notebook
app.py - Flask application for group 13
BAIS6050_group13.postman_collection - This is a postman export of a collection. The collection contains an example HTTP POST and GET (test observation) for the given flask application.

NOTE: It is assumed that the notebooks and flask application will be run in the same directory.  This directory will contain a directory called "data"
